About

TL;DR Summarizer is a simple single-page web app developed using the Flask framework that performs automatic summarization of text and web pages.

The app showcases some of the basic capability of the PyTLDR module for Python, which is available as open source. It offers three distinct summarization algorithms based on academic research from the past decade. You can read more about these algorithms and their implementation here.
